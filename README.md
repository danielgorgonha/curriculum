Repository for my HTML resume: single master version that can be customized per job application.

---

## Structure

| File | Purpose |
|------|---------|
| [index.html](index.html) | Master resume (complete version, ATS-optimized) |
| [resume.pdf](resume.pdf) | PDF version for applications (regenerate after HTML updates) |
| [post.md](post.md) | Job description template for ATS optimization |
| [CHANGELOG.md](CHANGELOG.md) | Version history and notable changes |

---

## Workflow

1. **Keep master updated:** Edit `index.html` with experience, skills, and metrics
2. **Per application:** Copy job description into `post.md`
3. **Customize (optional):** Follow guide at `personal-development-index/brag/RESUME-CUSTOMIZATION-GUIDE.md`
4. **Export PDF:** Open `index.html` in browser, Print to PDF (Ctrl+P)
5. **Apply:** Use generated PDF for application
